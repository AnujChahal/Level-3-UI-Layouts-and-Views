# Level-3-UI-Layouts-and-Views

This project demonstrates **UI Layouts and Views in Android** using **Jetpack Compose** and traditional XML-based layouts.  
The highlight of this level is building a **BMI Calculator App** with dynamic results, showcasing how different layouts and views can be combined effectively.

---

## üìå Level 3: UI Layouts & Views

### Jetpack Compose Basics
- Introduction to declarative UI programming.
- Composables for building reusable UI components.
- State management with `remember` and `mutableStateOf`.

### Layout Types
- **Linear Layout**: Arranges elements vertically or horizontally.
- **Constraint Layout**: Flexible positioning with constraints.
- **Frame Layout**: Stacks elements on top of each other.
- **Relative Layout**: Positions elements relative to each other.

### Common Views
- **Buttons**: Trigger actions on user interaction.
- **TextView**: Display text content.
- **ImageView**: Show images and icons.
- **ScrollView**: Enable scrolling for long content.

---

## üì± App: BMI Calculator
- Input: Height and Weight.
- Output: Dynamic BMI result with category (Underweight, Normal, Overweight, Obese).
- UI: Combines layouts and views to create a clean, interactive interface.
- Demonstrates handling user input, calculations, and updating UI dynamically.

- Understanding different Android layouts and their use cases.
- Managing multiple views in a single screen.
- Handling user input and updating UI dynamically.
- Building a functional app that integrates layouts and logic.
